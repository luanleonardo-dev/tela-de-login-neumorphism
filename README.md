# 📖 Descrição

Este projeto consiste em uma interface de tela de login moderna utilizando HTML, CSS e JavaScript puro.
O foco principal é a aplicação do estilo visual Neumorphism e a implementação de alternância de tema (Light/Dark Mode) de forma simples e eficiente.

## 🎯 Objetivo

Demonstrar habilidades fundamentais em:

Estruturação de páginas com HTML5
Estilização avançada com CSS3
Manipulação do DOM com JavaScript
## 🧱 Tecnologias Utilizadas
HTML5
CSS3 (Flexbox + Neumorphism)
JavaScript
## 🖥️ Estrutura do Projeto
/projeto
│── index.html
│── styles.css
## ⚙️ Funcionalidades
### 🔐 Interface de Login
Campo de usuário
Campo de senha
Botão de login
### 🎨 Design Neumorphism
Uso de box-shadow duplo para efeito de profundidade
Aparência suave e moderna
### 🌙 Dark Mode
Alternância de tema via botão
Mudança dinâmica de cores com JavaScript
Aplicação via classe global (dark-mode)
